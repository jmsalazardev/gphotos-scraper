# gphotos-scraper

A tool to extract photos metadata from an google photos shared album.


- public url
- filename
- description
- file size
- image size
- creation date

## Steps
Install:

`npm install -G gphotos-scraper`

Usage:

`gphotos-scraper extract https://photos.app.goo.gl/hTBHnwByt4MDwxo66 ./data/file.json`
